ACT6-C2. Exposición y Consumo de APIs de Terceros

Este repositorio es la el frontend de la practica de Consumo de APIs de Terceros el cual se encarga de levantar el proyecto con Docker compose.

Para que este proyecto pueda arrancar a la perfeccion tienen que contar con lo siguiente 

1-clonar el repositorio de la infra y crear un archivo .env en la rais de este proyecto para colocar las variables de la API 
se pueden guiar de el archivo .env.example del proyecto para colocar las variables 
2-tener instalado correctamente Docker Desktop

Una ves con los preparativos listos ya pueden levantar el proyecto colocando en el bash de la infra:

"docker compose up -d"

Listo ya puedes visualizar el proyecto en el localhost:3000
